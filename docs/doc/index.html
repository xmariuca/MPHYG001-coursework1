Welcome to Greengraph’s documentation!¶

This package measures how much green space there is between two locations. It outputs a graph with the distribution of green pixels from the discretized distance.

The package was created as part of Coursework 1 for the module MPHYG001: Research Software Engineering with Python.

greengraph.getGraph.plotGreenDistribution(startPos, endPos, steps, outFile)[source]¶

Generate a plot with the distribution of green pixels between two locations.

Parameters:
  • startPos (string) – A starting location, like ‘London’
  • endPos (string) – An ending location, like ‘Cambdrige’
  • steps (int) – A number of steps to discretise the space between the two locations, like 15
  • outFile (string) – A filename to be used for saving the plot, like ‘outGraph.png’
Returns:

A file with the graph representing the distribution of green pixels.
